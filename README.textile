h1. Riot TextMate bundle

A simple TextMate bundle for the Riot ruby unit testing framework

Does this code helps your daily work with Ruby? So, "please recommend me in Work With Rails":http://workingwithrails.com/recommendation/new/person/9370-lucas-h-ngaro and thanks for your kindness! :)

h2. Installation

<pre>
  cd ~/Library/Application\ Support/TextMate/Bundles
  git clone git://github.com/lucashungaro/riot-tmbundle.git Riot.tmbundle
  osascript -e 'tell app "TextMate" to reload bundles'
</pre>

Have fun!
